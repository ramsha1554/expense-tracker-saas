# ExTrack. - Premium Expense Tracker Dashboard

![Dashboard Preview](https://via.placeholder.com/1200x600.png?text=ExTrack+SaaS+Dashboard+Preview)

ExTrack is a production-ready, SaaS-style Expense Tracker Dashboard built with React and Tailwind CSS. It features real-time financial insights, advanced analytics, and a premium fintech-inspired UI.

## 🚀 Features

- **Smart AI Insights:** Automatically analyzes your spending patterns and provides actionable financial advice.
- **Real-Time Analytics:** Interactive charts for income vs. expenses and category-based breakdown using Recharts.
- **Multi-Currency Support:** Fully localized for Indian Rupees (INR) with proper formatting.
- **Transaction Management:** Full CRUD operations with modals, validation, and categories.
- **SaaS Layout:** Professional sidebar navigation, top navbar with user profile (Syed Ramsha), and responsive design.
- **Persistence:** LocalStorage integration ensures your data stays synced across sessions.
- **Smooth UX:** Framer Motion animations and hover effects for a premium feel.

## 🛠️ Tech Stack

- **Frontend:** React.js (Vite)
- **Styling:** Tailwind CSS
- **Icons:** Lucide React
- **Charts:** Recharts
- **Animations:** Framer Motion
- **State Management:** Context API
- **Date Handling:** date-fns

## 📦 Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/ramsha1554/expense-tracker-saas.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
